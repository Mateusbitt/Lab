Lab On - nullshCoalescing

Para o JS valores como 0, '', [], false, undefined, null => São considerados Falsy... ou seja não válido quando trazemos para o operador OU => ||.



Ai que entra o operador "nullshCoalescingOperator" (??). Ele é mais restritivo, ele vai olhar realmente para valores não significativos, como por exemplo o ZERO (0), ele tem sim um valor mesmo que primitivo.


--------------------------------------------------------------------------------------------------------


const age = 25

document.getElementById("example").innerHTML = "Sua idade é: " + (age ?? 'Não informado')

Sua idade é: 25


--------------------------------------------------------------------------------------------------------

const age = 0

document.getElementById("example").innerHTML = "Sua idade é: " + (age ?? 'Não informado')

Sua idade é: 0


--------------------------------------------------------------------------------------------------------

const age = null

document.getElementById("example").innerHTML = "Sua idade é: " + (age ?? 'Não informado')

Sua idade é: Não informado
